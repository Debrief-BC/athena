# Debrief

## Palomino
A web chat based on Debrief's Blockchain Protocol

## Setup Instructions
Checkout the repository
```
git clone https://github.com/Debrief-BC/palomino.git
```
Install dependencies
```
npm i
```
Start the front end server
```
npm run serve
```
